Loom Software Dependency Master List¶

This document contains all the open source and third party softwares that VxTitan (aka Loom) is dependent upon.

For users of Loom Azure SaaS deployment, only the components listed in the software dependency list is relevant.

Software Dependency for Loom SaaS Deployment Usage¶

Loom supports cloud as well as on premises deployment. For the initial release of Loom, only Microsoft Azure Cloud based SaaS deployment managed by Veritas is available for customers. To use Loom, customers need not deploy any additional software components if they wish to discover, visualize and scan content repositories existing in any of the cloud supported by the Loom Connector framework.

However, if customers wish to discover, visualize, scan and classify content repositories that are located on premises, they need to verify and ensure the software components and dependencies for Loom are met. The table below lists all such dependencies for Loom Alpha2 release.

Note

This section may be updated in subsequent releases of Loom.
